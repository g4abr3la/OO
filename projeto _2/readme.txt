

## Como Usar

1. Pré-requisitos: Certifique-se de ter o Python instalado no seu sistema.

2. Clonando o Repositório (ou baixe o arquivo diretamente):
   ```bash
   git clone https://github.com/seu-usuario/calculadora-areas-geometricas.git

1. Navegando até o Diretório:

bash
Copy code
cd calculadora-areas-geometricas

2. Executando o Programa:

bash
Copy code
python cod.py

3. Escolhendo a Forma Geométrica: O programa irá solicitar que você escolha uma forma geométrica digitando um número (1 para círculo, 2 para triângulo, 3 para retângulo).

4. Fornecendo Valores: Siga as instruções na tela para fornecer os valores necessários para calcular a área da forma geométrica selecionada.

5. Exibição do Resultado: O programa calculará a área e a exibirá no terminal.

6. Repetição: Você pode repetir o processo quantas vezes desejar.

Estrutura do Código
O código consiste em três classes, uma para cada forma geométrica, cada uma com um método para calcular a área.

A função main() inicia o programa e orienta o usuário a escolher a forma geométrica e fornecer os valores necessários.
